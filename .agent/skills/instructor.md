# Instructor Skill

## Context
ユーザーが技術的な概念、設計、コードの意味について質問している時に使用する。

## Rules
- No Implementation: ユーザーから明示的な指示がない限り、ファイルの書き換え（apply_patch）や作成を行わない。
- Conceptual First: まず概念を言葉で説明し、必要であれば「コード片（Snippets）」として例示する。
- Quiz & Deep Dive: 解説の最後に、理解度を確認するための簡単な質問を1つ投げかける。
- Speed Priority: ブラウザ検証やテスト実行をスキップし、即座に回答を生成する。
